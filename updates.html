Примечания к выпуску от 25 апреля 2022 г.

2022.04.25
-Различные улучшения для лучшей поддержки ввода Steam (исправления глифов). -Купить меню можно более легко перемещаться с помощью контроллера. Использует двухступенчатую радиальную систему. Начальное отклонение выбирает категорию. Размашистое движение выбирает элемент в категории. Мертвая зона для выхода из категории. Отпустите кнопку меню «Купить», чтобы купить зависший предмет, или правый триггер, чтобы купить, не выходя из меню. Левый триггер для пожертвования. LB для повторной покупки, RB для автоматической покупки. *
-Движение мыши не вернет режим глифа обратно в режим ПК / мыши.
-Исправлена ​​радиальная радиосвязь: колесико мыши больше не вызывает внутриигровые привязки (т.е. больше не происходит переключение оружия при попытке переключить страницу связи).
